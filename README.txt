Проект разбит на 5 модулей:
• app - главный модуль
• dictionary - модуль фича
• network, network-components, ui-components - общие модули

Использованы следущие библиотеки:
• Koin - используется для dependency injection
• Glide - для загрузки фотографий
• Retrofit - для работы с http
• Navigation graph - для навигаций фрагментов
• Timber - для логирования
• Coroutines - для ассинхронного программирования
